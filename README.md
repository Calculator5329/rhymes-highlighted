# Rhymes Highlighted

A specialized text editor for analyzing and highlighting rhyming patterns in poetry and song lyrics.

## What It Is

Rhymes Highlighted is a dedicated editor designed for poets and songwriters to write and analyze rhyming patterns in their work. The application visually highlights rhyming words in different colors, making it easy to see rhyme schemes at a glance. It supports saving multiple projects locally, importing/exporting work, and provides keyboard shortcuts for quick saving.

## Features

- **Rhyme Highlighting** - Automatically detects and color-codes rhyming patterns
- **Project Management** - Save and manage multiple projects locally
- **Import/Export** - Import projects from JSON files and export your work
- **Local Storage** - All projects saved to browser storage
- **Keyboard Shortcuts** - Ctrl+S to save quickly
- **Project Metadata** - Track creation and modification dates for each project
- **Clean Editor Interface** - Focused writing experience with minimal distractions

## Tech Stack

- React 19 with TypeScript
- Vite for development and building
- MobX for state management
- Tailwind CSS for styling
- LocalStorage API for project persistence

## Getting Started

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`

## Build
